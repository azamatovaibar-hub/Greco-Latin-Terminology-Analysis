**Supplementary material for the article submitted to the "Terminology" journal.**
From Convergence to Asymmetry: A Distributional Analysis of Greco-Latin Terminological Elements in Kazakh, English, and Russian Lexicographical Canons

This repository contains the Python code and supplementary materials for the lexical analysis presented in the article submitted to the **Terminology** journal. This repository is provided for peer-review purposes.

## Overview
The goal of this project is to provide a reproducible workflow for the distributional analysis of Greco-Latin terminological units across three languages: Kazakh, English, and Russian. 

The provided scripts generate the visualizations and statistical calculations discussed in the paper.

## Repository Structure
- `notebooks/`: Contains the Jupyter Notebook (`.ipynb`) with the complete analysis and visualization code.
- `data/`: Dataset used for the analysis.
- `results/`: High-resolution versions of the charts and graphs.

## How to Use
To reproduce the findings:
1. Open the notebook in the `notebooks/` directory.
2. You can run this notebook directly in [Google Colab](https://google.com) or locally using Jupyter.
3. The code is documented to explain each step of the terminological extraction and plotting process.
